# 🚀 Go后端开发工程师学习路径

> 专注于Go语言后端开发的系统化学习指南

## 🎯 岗位技能图谱

### 核心技能（必备）
- ✅ **Go语言基础**: 语法、并发、内存管理
- ✅ **Web框架**: Gin、Echo、Fiber框架使用
- ✅ **数据库**: MySQL、PostgreSQL、MongoDB
- ✅ **缓存**: Redis、Memcached
- ✅ **消息队列**: Kafka、RabbitMQ、NATS

### 进阶技能（加分）
- 🌟 **微服务**: gRPC、服务发现、链路追踪
- 🌟 **性能优化**: profiling、内存泄漏排查
- 🌟 **容器化**: Docker、K8s基础
- 🌟 **监控**: Prometheus、Grafana、日志收集
